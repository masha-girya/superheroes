# Frontend

## Technologies
Typescript
React
Router
MUI
Formik

## Run the project
1. clone repository and open it in a program
5. run `npm install`
3. run `npm start`
4. enjoy!

## Routing
### main page
`http://localhost:3000/`

### superhero page
`http://localhost:3000/#/NICKNAME`
where NICKNAME is param and is superhero's nickname

### edit superhero
`http://localhost:3000/#/edit/NICKNAME`
where NICKNAME is param and is superhero's nickname

### add a superhero
`http://localhost:3000/#/add`
